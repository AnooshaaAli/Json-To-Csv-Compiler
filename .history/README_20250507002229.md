# JSON TO CSV COMPILER

## Pre requisites
```sh
brew install flex bison
```

---

## How to compile?
### scanner
```sh
flex scanner.l
gcc -o scanner scanner.c
./scanner < input.json
```

### parser
```sh
bison -d parser.y
flex scanner.l
gcc -o json2relcsv main.c parser.tab.c lex.yy.c ast.c -ly -ll
./json2relcsv --print-ast < input.json 
```
